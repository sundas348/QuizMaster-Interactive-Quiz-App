# 🧠 QuizMaster — Interactive Quiz App

A premium, interactive quiz application built with **HTML, CSS, and JavaScript** featuring a modern glassmorphism UI, dark mode, timer, and leaderboard.

## ✨ Features

- 🎯 **Multiple Categories** — Science, Technology, History, Mathematics, General Knowledge
- 📊 **Difficulty Levels** — Easy, Medium, Hard
- ⏱️ **Timed Questions** — 30 seconds per question with visual countdown
- 🔥 **Streak System** — Bonus points for consecutive correct answers
- 🌙 **Dark Mode** — Toggle between light and dark themes
- 🏆 **Leaderboard** — Saves top scores in localStorage
- 📋 **Answer Review** — Review all answers after the quiz
- 📱 **Fully Responsive** — Works on all devices

## 🛠️ Tech Stack

- HTML5
- CSS3 
- Vanilla JavaScript (ES6+)

## 🎯 How to Play

1. Select a **category** (or "All Topics")
2. Choose a **difficulty** level
3. Pick the **number of questions**
4. Enter your **name**
5. Click **"Start Quiz"**
6. Answer each question within **30 seconds**
7. View your **score, streak, and review answers**
8. Save your score to the **leaderboard**
